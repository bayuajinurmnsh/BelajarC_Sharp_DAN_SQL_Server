Panduan Penggunaan:

1. Buka xampp dan start MySQL dan Apache(optional)
2. Buka Aplikasi di visual studio / vs code dan ketik dotnet run

3. buka url https://localhost:5001 atau https://localhost:5001/api/OwnerDetails
   jika menggunakan swagger buka https://localhost:5001/swagger/index.html

3a. buka [URL] https://localhost:5001/api/AuthManagement/Register
    ganti method menjadi [POST] dan pastikan username,email,dan password terisi lalu tekan enter/execute

3b. buka [URL] https://localhost:5001/api/AuthManagement/Login
    ganti method menjadi [POST] dan pastikan username,dan password terisi lalu tekan enter/execute 
    

3c. buka [URL] https://localhost:5001/api/AuthManagement/RefreshToken
    ganti method menjadi [POST] dan copykan Token dan Refresh Token yang didapat setelah Login

4. buka [URL] https://localhost:5001/api/OwnerDetails
   ganti method menjadi [GET], tekan enter/execute

5. buka [URL] https://localhost:5001/api/OwnerDetails
   ganti method menjadi [POST] dan isi field pada ownerId, ownerCardNumber. ownerName, ownerAddress, 
   status, dan CreatedDate
   lalu klik execute/enter

6. buka [URL] https://localhost:5001/api/OwnerDetails/id
   ganti method menjadi [GET], ganti id dengan id yang dimiliki ownerId
   lalu klik execute/tekan enter

7. buka [URL] https://localhost:5001/api/OwnerDetails/id
   ganti method menjadi [PUT], ganti id dengan id yang dimiliki ownerId
   lalu isikan ownerId dengan ownerId user yang akan diganti, ownerCardNumber, ownerName, 
   ownerAddress, status, dan createdDate
   lalu tekan enter/ klik execute

8. buka [URL] https://localhost:5001/api/OwnerDetails/id
   ganti method menjadi [DELETE], ganti id dengan id yang dimiliki ownerId
   lalu tekan enter/ klik execute
